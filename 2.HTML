Web vs Internet

Web

É um dos serviços que roda sobre a internet. A web consiste em páginas da web e aplicativos acessíveis através de navegadores. Ela utiliza o protocolo HTTP (ou HTTPS) para transferir dados e é organizada por meio de hyperlinks. A web é, portanto, uma parte da internet, mas não a totalidade dela.

Internet

É uma vasta rede global de computadores e servidores interconectados. Ela permite a comunicação entre dispositivos e abrange uma variedade de serviços, como e-mail, transferência de arquivos e redes sociais. Em essência, a internet é a infraestrutura que conecta todos esses sistemas.
